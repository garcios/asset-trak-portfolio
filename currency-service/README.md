# Currency Service


## Install toml parser
```shell
go get github.com/BurntSushi/toml
```

## Install my custom libraries
```shell
go get github.com/garcios/asset-trak-portfolio/lib@main 
```

## Set env variables
```shell
export DBUSER=root
export DBPASS=Pass123
```
